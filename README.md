A Simple Perceptual Reference Method for Reducing False-Positive UAP Observations -- The STAR Method (Stable Thumb Alignment Reference) - v1.09

This simple observational reference method may help reduce false or misidentified UAP sightings before reports are submitted. Included is text which might accompany the image:
The STAR Method

"Stable Thumb Alignment Reference"

Basic skywatching advice.

After staring at a bright object in the sky for a few seconds, stationary stars can suddenly appear to move or dart about in darkness due to what is called the autokinesis effect.  Without a nearby stable reference, the visual system can manufacture convincing apparent motion.

So, when you see an object in the sky that you don't know what it is, always first check against a fixed point to ensure that it isn't a star, planet or airplane.

The idea behind the STAR Method is intentionally very simple: 

Before recording a distant point-light source in the night sky, the observer briefly uses their extended thumb as a stable nearby angular reference point. 

This may help distinguish genuine external motion from internally generated apparent motion caused by effects such as autokinesis, microsaccades, atmospheric scintillation, hand tremor amplification, or smartphone stabilization artifacts.

     1. Extend Thumb, Brace Arm Firmly
     2. Align UAP at End of Thumb
     3. Keep Hand Steady
     4. Reference, Is UAP Moving?
